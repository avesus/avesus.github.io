Built Mechanisms And Evidence

Start with the mechanism, then inspect the boundary of the claim.

I reconstruct parts of computing that are usually hidden behind an inherited stack: arithmetic schedules, GPU state transitions, configuration paths, routing, timing, and physical process. The strongest public work keeps the circuit, source, run, and limitation close enough for another engineer to challenge or extend it.

This is the public workbench, not a substitute for independent verification or silicon signoff.

Three Places To Start

Arithmetic scheduling

Bubbles-Free Bit-Serial Multiplier

The public Logisim artifact is an unsigned positive-number 8x8 multiplier with an 8-bit LSB-first result, three clocks of fixed latency, and one result bit per clock after fill. Consecutive operand words do not require drain bubbles.

Boundary: a circuit and composable schedule, not a measured claim that it outperforms inferred multipliers or FPGA DSP blocks.

Browser GPU systems

WebGL And GLSL As A Computing Substrate

Raw WebGL1 and GLSL ES 1.00 work exposes context setup, shader compilation, packed texture state, framebuffer passes, ping-pong updates, and readback across cellular automata and reconfigurable-fabric experiments.

Boundary: direct implementation evidence, not a cross-browser performance benchmark or a claim that browser simulation is hardware.

Reconfigurable computation

Learn Cartilage, One Chapter At A Time

The ordered path begins with Logisim and LUTs, then builds through CMOS gain, output drivers, clock and event distribution, state machines, timing, metal routing, nested components, and runtime reconfiguration in space.

Boundary: the path distinguishes teaching models, browser artifacts, matched RTL examples, and still-open device research.

FPGA Systems Evidence

The FPGA systems map connects work that was previously scattered across radio, arithmetic, archives, and source files.

RF transmit and receive

One-Pin Quadrature/SDM Transmitter

The transmitter wrapper traces two one-bit control paths through four carrier phases to one HX8K output and a resonant network. The companion one-pin FM receiver has operating video and a selected UP5K source trail. Fresh transmitter measurements and receiver calibration remain open.

Ethernet and configuration

Ethernet To An Adjacent iCE40

The 2020 multi-file source implements RMII receive, ARP response, fixed-layout IPv4/UDP-carried image ingestion, RAM storage, and direct iCE40LP384 configuration, plus a Linux UDP/XCB/EGL control shell.

Physical-computing education

Physical MUX Tiles

Original photographs establish fabricated role variants, connector-spaced palm-scale assemblies, and manual composition. The proposed STEM/EE path connects the tangible selector alphabet to Logisim and FPGA LUTs without claiming powered operation or classroom validation.

Current Cartilage Proof Package

Runnable bounded proofCartilage Core: Reconfigurable Fabric in WebGL and SystemVerilog

The public source repository publishes the compact WebGL1 model, editable hardware paper and PDF, local-clocked SystemVerilog RTL, and self-checking Verilator testbench. The checked 6x6 image receives 252 payload bits plus one apply pulse and settles to an interior-MUX AND circuit.

Boundary: this proves a logical mechanism and one exact installed image. It does not establish silicon signoff, a general compiler, or a general placer/router.

Runtime Instantiation in Space shows a bounded daughter region receiving a serial configuration stream from a local parent port. Cartilage Visual Language documents the current body-role alphabet.
