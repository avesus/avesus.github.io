How To Read Them

Checked runs and sketches do different work. A checked run can state frame counts, source files, visible output lanes, and expectation results. A visual sketch can explain mapping vocabulary, cell roles, and intended routing, but it should not be read as a driven circuit unless the page says so.

That distinction keeps the Cartilage material useful. The generated pages can show artifact paths and design vocabulary without blurring a sketch into a completed verification run.
