How To Read Them

Cartilage Core is the current compact implementation and hardware reference. It is not the source tree for the older QuadFlow/QFG captures listed below.

Checked runs and sketches do different work. A checked run can state frame counts, source files, visible output lanes, and expectation results. A visual sketch can explain mapping vocabulary, cell roles, and intended routing, but it should not be read as a driven circuit unless the page says so.

That distinction keeps the Cartilage material useful. The pages can show artifact paths and design vocabulary without blurring a sketch into a completed verification run.
